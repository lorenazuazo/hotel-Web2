
* HTML es el lenguaje con el que se “escriben” las páginas web.
- Basado en etiquetas:
	Texto/Imagenes/Tablas/Formularios.
- Se fueron agregando:
	Applets Java.
	Texto fluyendo alrededor de imágenes.
	Hojas CSS.
	Scripts (Código de aplicaciones) -- Páginas Dinámicas.
	
XHTML =  Adaptación de HTML al lenguaje XML (eXtensible Markup Language).

Incluir en una misma página HTML los contenidos, el diseño y la programación complica en exceso 
su mantenimiento. Normalmente, los contenidos y el diseño de la página web son responsabilidad
de diferentes personas, por lo que es conveniente separarlos.

CSS =  Mecanismo que permite separar los contenidos definidos mediante XHTML y el aspecto 
que deben presentar esos contenidos.

Etiquetas:
- El proceso de indicar las diferentes partes que componen la información se denomina marcar (markup en inglés). 
  Cada una de las palabras que se emplean para marcar el inicio y el final de una sección se denominan Etiquetas.
- En general las etiquetas se indican por pares y se forman de la siguiente manera:  
  Etiqueta de apertura: carácter <, seguido del nombre de la etiqueta (sin espacios en blanco) y terminado con el carácter >.
  Etiqueta de cierre: carácter <, seguido del carácter /, seguido del nombre de la etiqueta (sin espacios en blanco) y 
  terminado con el carácter >.
- Se pueden personalizar las etiquetas HTML mediante cierta información adicional llamada Atributo.  
  Los atributos se incluyen dentro de la etiqueta de apertura.
  Cada etiqueta define su propia lista de atributos disponibles.
  Cada atributo también indica el Tipo de Valor que se le puede asignar.
  * Algunos atributos son comunes a muchas o casi todas las etiquetas.

Elementos HTML (en linea/de bloque):
- Etiqueta apertura.
- Cero o mas atributos.
- Texto encerrado en etiqueta.
- Etiqueta de cierre.

Párrafos (etiqueta): <p></p>
	El navegador se encarga de ajustar su longitud al tamaño de la ventana.
	El navegador también se encarga de separar automáticamente cada párrafo de distinta etiqueta.

Secciones (jerarquicas): <h1>, <h2>, <h3>, <h4>, <h5> y <h6>
	Definen títulos de sección, no secciones completas.
	Seis niveles de importancia.
	Los navegadores asignan de automáticamente el tamaño del título de cada sección en función de su importancia.

Marcado de texto:
	<em> en cursiva.
	<strong> en negrita.

Nuevas lineas:
	<br/> (abreviatura) forzar a que el texto que sigue se muestre en la línea inferior.
		Indica que una etiqueta se está abriendo y cerrando --> <br></br>

Espacios en blanco: 
	&nbsp; se debe sustituir cada nuevo espacio en blanco.

Enlaces:
	absoluto (externo) --> acceder a páginas que se encuentran en otros sitios. 
	relativo (interno) --> apuntan a páginas del propio sitio web -versión abreviada de una URL absoluta-.

Enlaces básicos:
	<a>  --> <a href=“http://www.google.com”>Página principal de Google</a> 
	href --> atributo que indica la URL a la apunta el enlace.

Listas:
	No ordenadas --> solo relaciona los elementos.
		<ul> encierra todos los elementos de la lista y la etiqueta <li> cada uno de sus elementos.
	Ordenadas    --> enumera elementos.
		<ol> encierra todos los elementos de la lista y la etiqueta <li> cada uno de sus elementos.

Imágenes:
	Adorno (CSS)    --> iconos, fondos de pagina, etc.
	Contenido <img> --> proporcionan informacion y complementan informacion textual.
		src = “url” - Indica la URL de la imagen que se muestra.
		alt = “texto” - Descripción corta de la imagen.
		longdesc = “url” - Indica una URL en la que puede encontrarse una descripción más detallada de la imagen. 
		name = “texto” - Nombre del elemento imagen.
		height = “unidad_de_medida” - Indica la altura con la que se debe mostrar la imagen (no es obligatorio que coincida con la 			altura original de la imagen).
		width = “unidad_de_medida” - Indica la anchura con la que se debe mostrar la imagen (no es obligatorio que coincida con la 		anchura original de la imagen). 	
	<img src=”logotipo.gif” alt=”Logotipo de Mi Sitio” /> (etiqueta vacía, se cierra en la misma)

Tablas:
	Las tablas de HTML pueden contener elementos simples, agrupaciones de filas y de columnas, cabeceras y pies de tabla, subdivisiones, 	cabeceras múltiples y otros elementos complejos.

Tablas básicas:
	Las tablas más sencillas de HTML se definen con tres etiquetas: <table> para crear la tabla, <tr> para crear cada fila y <td> para 	crear cada columna.  
	Ver: Atributos/Tablas anidadas.
